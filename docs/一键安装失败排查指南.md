## 🔍 常见失败原因

### 1. Python未安装或未添加到PATH

**错误信息**:
```
❌ Python未安装或未添加到PATH
'python' 不是内部或外部命令
```

**解决方案**:

#### 步骤1: 下载Python
1. 访问 https://www.python.org/downloads/
2. 下载 Python 3.10 或 3.11（推荐3.11）
3. 不要下载3.12+，可能有兼容性问题

#### 步骤2: 安装Python（关键！）
1. 运行安装程序
2. **务必勾选** "Add Python to PATH"（第一页底部）
3. 选择 "Install Now"
4. 等待安装完成

#### 步骤3: 验证安装
打开命令提示符（CMD），输入：
```cmd
python --version
```

如果显示版本号，说明安装成功。

#### 步骤4: 重启CMD
安装Python后，**必须重启命令提示符**才能生效。

---

### 2. pip不可用

**错误信息**:
```
❌ pip不可用
'pip' 不是内部或外部命令
```

**解决方案**:

#### 方法1: 使用 python -m pip
如果pip命令不可用，但python -m pip可用：
```cmd
python -m pip install -r requirements.txt
```

#### 方法2: 重新安装Python
1. 卸载现有Python
2. 重新安装，确保包含pip
3. 安装时勾选所有选项

---

### 3. Node.js未安装或未添加到PATH

**错误信息**:
```
❌ Node.js未安装或未添加到PATH
'node' 不是内部或外部命令
```

**解决方案**:

#### 步骤1: 下载Node.js
1. 访问 https://nodejs.org/
2. 下载 **LTS版本**（推荐18.x或20.x）
3. 不要下载最新版（可能有兼容性问题）

#### 步骤2: 安装Node.js（关键！）
1. 运行安装程序
2. **务必勾选** "Add to PATH"（或"Automatically install the necessary tools"）
3. 选择 "Install"
4. 等待安装完成

#### 步骤3: 验证安装
打开CMD，输入：
```cmd
node --version
npm --version
```

#### 步骤4: 重启CMD
安装Node.js后，**必须重启命令提示符**。

---

### 4. 后端依赖安装失败

**错误信息**:
```
❌ 后端依赖安装失败
pip install -r requirements.txt
ERROR: Could not find a version...
```

**解决方案**:

#### 方法1: 使用国内镜像
```cmd
cd backend
python -m pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

#### 方法2: 逐个安装依赖
```cmd
cd backend
python -m pip install fastapi
python -m pip install uvicorn[standard]
python -m pip install sqlmodel
# ... 逐个安装 requirements.txt 中的包
```

#### 方法3: 升级pip和setuptools
```cmd
python -m pip install --upgrade pip setuptools wheel
```

#### 方法4: 安装Visual Studio Build Tools
某些Python包需要编译，需要：
1. 下载 Visual Studio Build Tools
2. 安装"使用C++的桌面开发"组件

---

### 5. 前端依赖安装失败

**错误信息**:
```
❌ 前端依赖安装失败
npm ERR! network...
npm ERR! code E404
```

**解决方案**:

#### 方法1: 使用国内镜像
```cmd
cd frontend
npm config set registry https://registry.npmmirror.com
npm install
```

#### 方法2: 清除npm缓存
```cmd
npm cache clean --force
npm install
```

#### 方法3: 删除node_modules重试
```cmd
cd frontend
rmdir /s /q node_modules
npm install
```

#### 方法4: 使用淘宝镜像（备选）
```cmd
npm install --registry=https://registry.npmmirror.com
```

### 6. 权限问题

**错误信息**:
```
Permission denied
Access is denied
```

**解决方案**:

#### 以管理员身份运行
1. 右键点击安装脚本
2. 选择"以管理员身份运行"

#### 检查文件夹权限
1. 右键点击项目文件夹
2. 属性 → 安全
3. 确保当前用户有完全控制权限

---

### 7. 网络连接问题

**错误信息**:
```
ReadTimeoutError
Connection refused
Failed to establish a new connection
```

**解决方案**:

#### 使用国内镜像
**Python**:
```cmd
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
```

**npm**:
```cmd
npm config set registry https://registry.npmmirror.com
```

#### 检查防火墙和代理
1. 检查防火墙是否阻止了下载
2. 如果使用代理，配置代理设置
3. 尝试关闭VPN后安装

- frontend/

如果文件丢失，重新克隆或下载项目。
