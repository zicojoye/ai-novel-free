# 启动问题完整解决指南

## 问题1：端口占用

### 症状
```
Address already in use
```

### 解决方案
运行停止脚本：
```cmd
停止.bat
```

这会关闭所有后端和前端进程。

---

## 问题2：配置验证错误

### 症状
```
❌ 启动失败: 6 validation errors for Settings
Extra inputs are not permitted
```

### 原因
.env文件中的配置项在Settings类中没有定义

### 已修复
已在 `backend/app/core/config.py` 中添加缺失的字段：
- RATE_LIMIT_PER_MINUTE
- CSP_DEFAULT_SRC
- CSP_SCRIPT_SRC
- CSP_STYLE_SRC
- CSP_IMG_SRC
- CSP_CONNECT_SRC

---
