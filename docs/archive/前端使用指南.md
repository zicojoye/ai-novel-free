## 服务层使用

所有服务位于 `src/services/` 目录，提供统一的API调用接口。

### 项目服务

```tsx
import { projectService } from '@/services'

// 获取项目列表
const projects = await projectService.getProjects({ page: 1, pageSize: 10 })

// 获取项目详情
const project = await projectService.getProject(projectId)

// 创建项目
const newProject = await projectService.createProject({
  name: '我的小说',
  genre: '都市',
  targetAudience: '25-35岁男性',
})

// 更新项目
await projectService.updateProject(projectId, {
  description: '更新后的描述',
})

// 删除项目
await projectService.deleteProject(projectId)
```

### AI服务

```tsx
import { aiService } from '@/services'

// 生成世界观
const worldview = await aiService.generateWorldview({
  project_id: 1,
  genre: '都市',
  core_concept: '灵气复苏',
})

// 生成章节
const chapter = await aiService.generateChapter({
  project_id: 1,
  chapter_number: 1,
  outline: '第一章大纲',
})

// 润色文本
const polished = await aiService.polishText({
  text: '要润色的文本',
  style: '文学',
})
```

### 其他服务

## 状态管理

使用 Zustand 进行状态管理，支持开发工具和持久化。

### 认证状态

```tsx
import { useAuthStore } from '@/stores/authStore'

function MyComponent() {
  const { user, isAuthenticated, login, logout } = useAuthStore()

  const handleLogin = async () => {
    const { data } = await authService.login({
      email: 'user@example.com',
      password: 'password',
    })
    if (data) {
      login(data.user, data.token)
    }
  }

  const handleLogout = () => {
    logout()
  }

  if (!isAuthenticated) {
    return <button onClick={handleLogin}>登录</button>
  }

  return (
    <div>
      <p>欢迎, {user?.username}!</p>
      <button onClick={handleLogout}>退出</button>
    </div>
  )
}
```

### UI状态

```tsx
import { useUIStore } from '@/stores/uiStore'

function MyComponent() {
  const { theme, toasts, showToast, removeToast, toggleModal } = useUIStore()

  const handleSuccess = () => {
    showToast({
      type: 'success',
      title: '操作成功',
      message: '项目已创建',
      duration: 3000,
    })
  }

  const handleOpenModal = () => {
    toggleModal('createProject', true)
  }

  return (
    <div>
      <button onClick={handleSuccess}>显示成功提示</button>
      <button onClick={handleOpenModal}>打开对话框</button>
      {toasts.map((toast) => (
        <div key={toast.id}>
          {toast.title}
          <button onClick={() => removeToast(toast.id)}>关闭</button>
        </div>
      ))}
    </div>
  )
}
```
