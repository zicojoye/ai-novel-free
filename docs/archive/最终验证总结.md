## 🚀 立即开始

### 快速启动 (推荐)

```bash
# 1. 安装依赖
cd backend && pip install -r requirements.txt
cd frontend && npm install

# 2. 配置环境
cp .env.example .env
# 编辑.env,填入API Keys

# 3. 启动服务
# Windows
start.bat

# Linux/Mac
./start.sh
```
