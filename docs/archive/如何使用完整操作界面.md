## 启动完整界面

### 方法 1: 一键启动（推荐）

直接双击运行:
```
启动完整界面.bat
```

这个脚本会自动:
1. 检查环境（Python + Node.js）
2. 安装前端依赖（如果需要）
3. 启动后端服务（http://localhost:8000）
4. 启动前端服务（http://localhost:5173）
5. 自动打开浏览器

### 方法 2: 手动启动

#### 步骤 1: 启动后端

打开一个 CMD 窗口:
```cmd
cd e:\Git 库\AI小说项目\ai-novel-platform\backend
python main_fixed.py
```

等待看到:
```
INFO:     Uvicorn running on http://0.0.0.0:8000
```

#### 步骤 2: 启动前端

打开另一个 CMD 窗口:
```cmd
cd e:\Git 库\AI小说项目\ai-novel-platform\frontend
npm install      # 首次运行需要安装依赖
npm run dev
```

等待看到:
```
  VITE v5.0.8  ready in xxx ms

  ➜  Local:   http://localhost:5173/
```

#### 步骤 3: 访问界面

在浏览器中打开: http://localhost:5173

## 常见问题

### Q1: `'vite' 不是内部或外部命令`

**原因**: 前端依赖未安装

**解决**:
```cmd
cd e:\Git 库\AI小说项目\ai-novel-platform\frontend
npm install
```

### Q2: npm 安装依赖失败

**解决方法**:

1. **使用国内镜像**:
```cmd
npm config set registry https://registry.npmmirror.com
npm install
```

2. **清除缓存重试**:
```cmd
npm cache clean --force
npm install
```

3. **使用 --legacy-peer-deps**:
```cmd
npm install --legacy-peer-deps
```

### Q3: 端口被占用

**后端端口 8000 被占用**:
```cmd
netstat -ano | findstr :8000
taskkill /PID <进程ID> /F
```

**前端端口 5173 被占用**:
```cmd
netstat -ano | findstr :5173
taskkill /PID <进程ID> /F
```

或者修改 `frontend/vite.config.ts` 中的端口号。

### Q4: 前端无法连接后端

检查:
1. 后端服务是否运行: http://localhost:8000/health
2. API 代理配置: `frontend/vite.config.ts`
3. CORS 配置: 后端已配置允许所有来源
