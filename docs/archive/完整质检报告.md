---

### 🟡 P1 - 非阻塞性问题

#### 问题 1: 前端代理配置不完整

**位置**: `frontend/vite.config.ts:14-18`

**当前配置**:
```typescript
proxy: {
  '/api': {
    target: 'http://localhost:8000',
    changeOrigin: true,
  },
}
```

**问题**: 缺少路径重写规则

**修复**:
```typescript
proxy: {
  '/api': {
    target: 'http://localhost:8000',
    changeOrigin: true,
    rewrite: (path) => path.replace(/^\/api/, '/api'),
  },
}
```

**修复优先级**: P1（中）

---

#### 问题 2: CORS 配置安全性

**位置**: `backend/main_fixed.py:37`

**当前配置**:
```python
allow_origins=["*"],  # 不安全的生产配置
```

**问题**: 允许所有来源的跨域请求

**修复**:
```python
allow_origins=settings.BACKEND_CORS_ORIGINS,
```

**修复优先级**: P2（低 - 生产环境才需要）
