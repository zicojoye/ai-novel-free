## 🚀 快速启动

### 1. 项目初始化
```bash
cd ai-novel-platform
python init.py
```

### 2. 配置环境变量
```bash
cp .env.example .env
# 编辑.env,填入API Keys
```

### 3. 安装依赖
```bash
npm run install:all
```

### 4. 启动服务
```bash
npm run dev
```

### 访问
- 前端: http://localhost:5173
- 后端: http://localhost:8000
- API文档: http://localhost:8000/docs

## 🤖 Agent系统详情

### 核心创作组 (4个)

1. **AuthorAgent (主创Agent)**
   - 创建章节
   - 创建大纲

2. **EditorAgent (编辑Agent)**
   - 润色文本
   - 去除AI痕迹

3. **ReviewerAgent (审核Agent)**
   - 质量检查
   - 合规审查

4. **PublisherAgent (发布Agent)**
   - 多平台发布
   - 格式转换

### 专项支持组 (3个)

5. **WorldBuilderAgent (世界观Agent)**
   - 生成10维度世界观

6. **KnowledgeManagerAgent (知识Agent)**
   - 提取知识
   - 搜索知识

7. **SemanticRetrieverAgent (语义检索Agent)**
   - 语义检索
   - 提供上下文

### 质量保障组 (2个)

8. **LogicCheckerAgent (逻辑Agent)**
   - 逻辑一致性检查

9. **StyleCheckerAgent (风格Agent)**
   - 文风一致性检查
