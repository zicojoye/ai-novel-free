### 方案 C：安装系统级 Node.js（彻底解决）💡

**状态**: 需要用户操作

**步骤**:

#### 1. 下载 Node.js

访问官方站点：https://nodejs.org/

推荐下载 **LTS** 版本（如 v20.x）

#### 2. 安装 Node.js

1. 双击下载的 `.msi` 安装包
2. 勾选 "Automatically install the necessary tools..."
3. 使用默认路径：`C:\Program Files\nodejs`
4. 点击 "Next" 直到完成安装

#### 3. 验证安装

打开新的 CMD 或 PowerShell 窗口：

```cmd
node --version
npm --version
```

应该显示类似：
```
v20.10.0
10.2.3
```

#### 4. 安装前端依赖

```cmd
cd "e:\Git 库\AI小说项目\ai-novel-platform\frontend"
npm install
```

如果提示权限问题，以管理员身份运行 CMD。

#### 5. 启动开发服务器

```cmd
npm run dev
```

#### 6. 访问应用

打开浏览器访问：http://localhost:5173

**优点**:
- ✅ 彻底解决所有 npm 问题
- ✅ 可以在任何终端中使用
- ✅ 完整的开发体验
- ✅ 未来项目也可以使用

**缺点**:
- ❌ 需要下载安装（约 40MB）
- ❌ 需要几分钟时间

## 故障排查

### 问题 1：npm 命令不存在

**症状**: `'npm' 不是内部或外部命令`

**解决**:
- 安装系统级 Node.js（方案 C）

### 问题 2：npm install 失败

**症状**: 安装过程中出现错误

**解决**:
```cmd
# 1. 清理缓存
npm cache clean --force

# 2. 删除 node_modules
rmdir /s /q node_modules

# 3. 重新安装
npm install
```

### 问题 3：npm run dev 报错 "vite not found"

**症状**: `vite 不是内部或外部命令`

**原因**: 依赖未安装

**解决**:
```cmd
npm install
```

### 问题 4：端口被占用

**症状**: `Port 5173 is in use`

**解决**:
```cmd
# 方法1：关闭占用进程
netstat -ano | findstr :5173
taskkill /PID <进程ID> /F

# 方法2：使用其他端口
npm run dev -- --port 3000
```
