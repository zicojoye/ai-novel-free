## 🚀 下一步操作

### 启动项目

#### 方式1: Docker部署
```bash
cd ai-novel-platform

# 配置环境变量
cp .env.example .env
# 编辑.env,填入API Keys

# 启动服务
docker-compose up -d

# 访问
# 前端: http://localhost
# 后端: http://localhost:8000
# API文档: http://localhost:8000/docs
```

#### 方式2: 本地开发
```bash
cd ai-novel-platform

# 初始化
python init.py

# 安装依赖
npm run install:all

# 启动开发
npm run dev

# 访问
# 前端: http://localhost:5173
# 后端: http://localhost:8000
# API文档: http://localhost:8000/docs
```
