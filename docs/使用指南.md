# AI Novel Platform - 使用指南

## 快速开始

### 1. 启动项目
双击运行 `启动.bat`

脚本会自动：
- 检查环境（Python、Node.js）
- 首次运行安装依赖
- 验证配置（允许警告）
- 启动前后端服务

### 2. 访问界面
- **前端界面**: http://localhost:3000
- **后端API**: http://localhost:8000
- **API文档**: http://localhost:8000/docs

## 配置说明

### 首次启动时的警告

如果看到以下警告，不用担心：
```
[WARNING] Warnings:
  1. SECRET_KEY 使用了默认值
  2. 未配置任何 AI API 密钥
```

**这不会阻止启动**，你可以先查看界面，后续再配置。

### 配置 API 密钥

当需要使用AI功能时，编辑 `backend/.env` 文件：

```env
# 至少配置一个
OPENAI_API_KEY=sk-your-openai-key
# 或
ANTHROPIC_API_KEY=sk-ant-your-key
# 或
DEEPSEEK_API_KEY=sk-your-key
```

### 生成 SECRET_KEY

```cmd
cd backend
python generate_secret_key.py
```

然后复制生成的密钥到 `.env` 文件的 `SECRET_KEY` 字段。

## 停止服务

在后端和前端的窗口中按 `Ctrl+C`

## 环境要求

- **Python**: 3.8+
- **Node.js**: 18.x 或 20.x

## 常见问题

### Q: 启动时显示警告，怎么办？
A: 不影响使用。警告是为了提醒你配置API密钥，可以稍后配置。

### Q: 后续如何配置API密钥？
A: 编辑 `backend/.env` 文件，添加API密钥后重启项目。

### Q: 如何重启项目？
A: 先停止服务（按Ctrl+C），然后再次运行 `启动.bat`

### Q: 界面打不开怎么办？
A: 检查后端和前端窗口是否正常运行，确保没有错误信息。
