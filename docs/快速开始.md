# AI Novel Platform - 快速开始

## 环境要求

- Python 3.9+ (当前测试 Python 3.14)
- Node.js 18+

## 一键启动（Windows）

双击运行 `启动.bat`，脚本会自动：
1. 检测并安装后端/前端依赖
2. 启动后端服务（端口 8000）
3. 启动前端服务（端口 3000）

## 手动启动

### 后端

```bash
cd backend
pip install -r requirements.txt
python start.py
```

### 前端

```bash
cd frontend
npm install --legacy-peer-deps
npm run dev
```

## 访问地址

| 服务 | 地址 |
|------|------|
| 前端界面 | http://localhost:3000 |
| 后端 API | http://localhost:8000 |
| API 文档 | http://localhost:8000/docs |
| 健康检查 | http://localhost:8000/health |

## 配置 AI API 密钥

编辑 `backend/.env`，填入至少一个 API 密钥：

```env
OPENAI_API_KEY=sk-...
# 或
DEEPSEEK_API_KEY=sk-...
# 或
ANTHROPIC_API_KEY=sk-ant-...
```

生成安全密钥：
```bash
cd backend
python generate_secret_key.py
```

## 停止服务

双击 `停止.bat`，或在各服务窗口按 `Ctrl+C`。
